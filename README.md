# MLE Capstone Proposal: Arvato Customer Acquisition

Udacity Machine Learning Engineer capstone project proposal for the Arvato/Bertelsmann customer segmentation and campaign response prediction task.

## Proposal

The proposal is in `proposal.md` and covers all seven rubric points:

1. Domain Background  
2. Problem Statement  
3. Datasets and Inputs  
4. Solution Statement  
5. Benchmark Model  
6. Evaluation Metrics  
7. Project Design  

## Data

Data files (from Udacity / AZ Direct GmbH) are in the `data/` directory:

- `Udacity_AZDIAS_052018.csv` – General population
- `Udacity_CUSTOMERS_052018.csv` – Established customers
- `Udacity_MAILOUT_052018_TRAIN.csv` – Campaign targets with labels
- `Udacity_MAILOUT_052018_TEST.csv` – Campaign test set (Kaggle)

Data use is governed by AZ Direct GmbH terms; deletion required within 2 weeks of project completion.
